No need to buy a new router for World Wi-Fi. Most residential routers are compatible with our software.
A new user can connect to the system through an invite only. An invite is a referral link connected to the wallet of the user in the system. The working process of referral program is as follows: all wallets transfer 25% of their earnings, from ads shown on own router, to the users who provided an invite to connect to the system. Thus, router owners are motivated to expand the system wider. Each user is able to provide an invite and earn from ads shown on invited routers. No user will pay any amount to another user of the platform; the only source of all transactions between users is advertisement budget, which will be distributed among the wallets. First invites will be given to PRE-SALE/ TOKEN SALE participants and to the project team members to develop the network (root invite). A root invite is an analog of digital franchise with zero commission, which can be used to develop the project and get the maximum income. There are limited number of root invites, in case you do not wish to develop the project then you can sell it profitably to an active member of the community wishing to earn by developing the project. To connect to the system, a root invite acquired during the token sale or a basic invite given by another user should be applied. After launching the platform, anyone can get their basic invites on the World Wi-Fi website.

The gas is a unit of measurement in which the prices for calculations on the Etherium are measured. The more complex calculation is, the more it is rewarded with gas. This works as follows: the originator of a smart contract sets the price in gases. Miners, on whose computers the calculations are performed, decide whether to perform these calculations or not. If the price is too high, the calculations, and the contract, will never be executed.
